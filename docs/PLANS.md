## 플랜 개요

### 확정된 플랜 구조 (2-tier)

| 플랜 | 가격 | 핵심 차별점 |
|------|------|------------|
| **FREE** | $0/월 | 1 PC, 7일 이력, 200 메시지/일 |
| **PRO** | $6.99/월 | 무제한 PC, 무제한 이력, 무제한 메시지, 푸시 알림 |

> **TEAM 플랜 제거 결정**: Pocket AI의 핵심 가치는 "개인 개발자가 자신의 PC Claude 세션을 모바일에서 접근"하는 것. 팀 협업(한 사람의 세션을 여러 명이 보는 것)은 사용 시나리오가 불명확하고 기술 복잡성 대비 ROI가 낮음.

### 설계 원칙

1. **FREE는 충분히 유용**: 7일 이력 + 200 메시지/일이면 캐주얼 사용자 커버
2. **PRO 전환 동기 명확**: 장기 프로젝트, 다중 PC, 헤비 사용자는 자연스럽게 PRO
3. **불필요한 제한 제거**: 디바이스 제한 없음 (OAuth로 보안 확보)
4. **비용 효율적**: 아카이빙으로 무제한 이력도 저비용 유지

## 기능 매트릭스

### FREE 플랜 ($0/월)

| 기능 | 제한 | 이유 |
|------|------|------|
| PC 세션 | 1개 | 집 or 회사 PC 하나 |
| 디바이스 | **무제한** | OAuth 인증으로 보안 확보, 제한 불필요 |
| 일일 메시지 | **200개** | 캐주얼 유저 충분, 헤비 유저 PRO 전환 |
| **메시지 이력** | **7일 보관** | 단기 프로젝트 커버 |
| E2E 암호화 | 포함 | 핵심 기능 |
| 지원 | 커뮤니티 | 비용 절감 |

**200 메시지/일 선택 이유**:
- 100개는 활발한 개발 시 오후에 막힘 (짜증 유발)
- 200개면 대부분 하루 충분
- 정말 헤비한 사용자만 PRO 전환 → 타겟 정확

### PRO 플랜 ($6.99/월)

| 기능 | 제한 | 이유 |
|------|------|------|
| PC 세션 | 무제한 | 집 + 회사 + 노트북 |
| 디바이스 | 무제한 | 어디서든 접속 |
| 일일 메시지 | 무제한 | 제한 없음 |
| **메시지 이력** | **무제한 (아카이빙)** | 장기 프로젝트 완벽 지원 |
| 푸시 알림 | 포함 | 세션 완료/에러 알림 |
| 우선 지원 | 포함 | 빠른 응답 |

**$6.99 선택 이유**:
- Termius Pro ($9.99)보다 저렴 → 진입 장벽 낮춤
- Claude Pro ($20) + Pocket AI ($6.99) = $27/월 → 합리적
- 연간 결제 시 $59/년 (30% 할인)

## 스토리지 최적화 전략

### 1. FREE 플랜: 7일 만료 삭제

```
┌─────────────────────────────────────────────────────┐
│  FREE 플랜 메시지 라이프사이클                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Day 0     Day 7      삭제                          │
│  ──────────────────────→                            │
│  [메시지 생성] [보관 중] [cron job으로 삭제]          │
│                                                     │
│  스토리지: ~100MB/유저 (7일 * ~700KB/일)             │
│  (200 메시지 × 500B × 7일)                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**구현 방법**:
```sql
-- 매일 실행: 7일 지난 FREE 유저 메시지 삭제
DELETE FROM messages
WHERE session_id IN (
  SELECT s.id FROM sessions s
  JOIN users u ON s.user_id = u.id
  JOIN subscriptions sub ON u.id = sub.user_id
  WHERE sub.plan = 'free'
)
AND created_at < NOW() - INTERVAL '7 days';
```

### 2. PRO 플랜: 30일 이후 아카이빙

```
┌─────────────────────────────────────────────────────────────────┐
│  PRO 플랜 메시지 라이프사이클                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Day 0          Day 30           영구 보관                      │
│  ──────────────────────────────────────→                        │
│  [활성 저장]    [아카이브로 이동]  [압축 스토리지]                │
│  (빠른 조회)    (읽기 전용)        (온디맨드 복원)               │
│                                                                 │
│  활성 스토리지: ~15MB/유저 (30일)                                │
│  아카이브 스토리지: 별도 관리 (저비용)                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**아카이빙 방식: Cloudflare R2**
- 무료 egress, $0.015/GB/월
- 30일 지난 세션 → JSONL로 압축 → R2 업로드 → DB에서 삭제
- 조회 시: R2에서 다운로드 → 캐시 → 표시

### 3. 비용 예측 (아카이빙 적용 시)

| 규모 | 활성 DB | 아카이브 (R2) | 총 비용 |
|------|---------|--------------|--------|
| 1,000 유저 | ~1GB | ~5GB | $0.08/월 |
| 5,000 유저 | ~5GB | ~25GB | $0.38/월 |
| 10,000 유저 | ~10GB | ~50GB | $0.75/월 |
